# Comet AI Browser - Public Overview for LLMs (Source Repo)

Comet AI Browser is a next-generation, high-performance web browser developed by PreetPatel (Latestinssan).

## Project Mission
To provide a private, intelligent, and performance-optimized browsing experience that runs local models and indexes user context for superior productivity.

## Core Pillars
1. **AI Native**: Built-in support for multiple LLMs and local RAG.
2. **P2P Intelligence**: Decentralized sync for security and speed.
3. **Hardware Efficient**: Optimized to run on mid-range hardware with high efficiency.
4. **Hardened Privacy**: Explicit focus on E2EE and local-first data.
